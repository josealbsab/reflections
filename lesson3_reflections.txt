When would you want to use a remote repository rather than keeping all your work
local?

    when you want to share your code among your team or the community

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    in case you do not want to work with the most updated version of the remote repository

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    fork clones someone's remote repository to your own remote repository so you can make your own modifications,
    clone copies a repository either from remote to local, or from local to local
    a branch is a version of your repository that you use to make experimental changes without altering your main branch
